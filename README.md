# Vector

Небольшой проект с собственной реализацией динамического массива `Vector<T>`, похожего на `std::vector`.

## Реализованные функции

1. Конструктор по умолчанию и конструктор с размером;
2. Методы size() и capacity();
3. reserve() и resize();
4. push_back() и pop_back();
5. Доступ к элементу через operator[] и at() (с проверкой границ)
